Hello World(modified)

第一篇文章直接把学md之前的示例文章(Hello World.md)内代码展示以供自己学习。

1.一行里不停写，都是同一段。
2.一行最前加$可以贴代码上去。
3.##，###大概是序号大小。
4.VScode里红标代表要空一行出来，最后一行也要。
5.基础md语法：

$
$ More info: Writing
$
$ ### Run server
$
$ bash $ $ hexo server $
$
$ More info: Server
$
$ ### Generate static files
$
$ bash $ $ hexo generate $
$
$ More info: Generating
$
$ ### Deploy to remote sites
$
$ bash $ $ hexo deploy $
$
$ More info: Deployment

```
